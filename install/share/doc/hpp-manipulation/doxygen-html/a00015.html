Detailed Description

Device with handles.

As a deriving class of hpp::pinocchio::HumanoidRobot, it is compatible with hpp::pinocchio::urdf::loadHumanoidRobot

This class also contains pinocchio::Gripper, Handle and JointAndShapes_t
