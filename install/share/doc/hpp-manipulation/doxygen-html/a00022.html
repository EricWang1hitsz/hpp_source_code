Detailed Description

This class represents a foliation of a submanifold of the configuration space.

Such a foliation is defined by the two following functions:

  • a condition $f$ such that the submanifold is ${M}= {q {C} | f(q)=0 }$
  • a parametrizer $g$ such that the leaf of this foliation is a level set of $g$.
