static std::string hpp::core::Parameter::typeName ( Type  type)
static

Return the value as a castable value into the approriate type.

For instance,

Parameter v1(5.0); // v1 is of type double
Parameter v2(3); // v2 is of type int
double x1 = v1.value();
double x2 = v2.value();

The first assignment will succeed, while the second one will throw an exception. Return the name of the type
